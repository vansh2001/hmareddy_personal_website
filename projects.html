Projects

Here are a few of my projects I worked on. Feel free to browse around and let me know if you have any questions.

Multilevel Cache Simulator

ECE 463 Microprocessor Architecture,  C++


Built a multilevel cache simulator with L1 and L2 cache in C++. The program calculates, hit rates, miss rates and number of write backs.



github link

Autonomous/WIFI Controlled PCB Robotic car

ECE 306 Embedded Systems,  C++

Coded in C and built an IOT motorized car using MSP430, IR sensor, and motor drivers. Used Tx and Rx to transfer data and controlled it using a laptop over WIFI. The car also follows black and colored lines.

Superscalar Pipeline Simulator

ECE 463 Microprocessor Architecture,  C++

Built a dynamic instruction scheduler that can take in any size width (instructions) and simulate a pipeline using ROB, Issue Queue and Register Read. In the end it also calculates cycle per instructions.

github link

IOS Swift Book Trading App

SWIFT 2,   XCODE,  Personal Project

Built an app in Xcode using Swift 2 that can trade books. Created a backend with firebase to authenticate users, store pictures and other relevant info.

github link

Branch Predictor Simulator

C++,  ECE 463 Microprocessor Architecture

Built a bimodal and gshare branch predictor in C++ with a 2-bit predictor. The program is also able to calculate the branch misprediction rate.
